## Description

A page where a user can add notes, clear forms, and delete previous notes.

## Installation

npm i
<br>
Dependencies: express

## Executing program

After installation
<br>
node server.js
<br>
Open your [localhost:3001](http://localhost:3001)

Create, save, and delete notes
